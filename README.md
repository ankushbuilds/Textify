# Textify

**Textify** is a simple and powerful text utility web application built with **React** and **Vite**. It allows users to quickly manipulate, format, and analyze text for writing, editing, or productivity purposes.

---

## 🔹 Features

- Convert text to **Uppercase / Lowercase**
- Remove extra spaces
- Copy text to clipboard
- View **word and character count**
- Clean, responsive UI

---

## 🛠 Tech Stack

- **React** – Frontend UI library
- **Vite** – Fast development and build tool
- **JavaScript** – Functionality
- **CSS** – Styling
